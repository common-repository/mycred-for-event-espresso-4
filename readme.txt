== Description ==

Trusted by thousands, [Event Espresso](http://eventespresso.com/pricing/?ap_id=mycred) is the best WordPress event online registration and ticketing manager plugin–and the best supported with full-time support. Turn your existing blog or website into a fully-featured event management website and a new way to make money.

The myCred plugin has built-in support for Event Espresso 4.5 and earlier. Version 4.6 or higher however is only supported via this custom plugin.

= myCred for Event Espresso 4 Features =
Here are the most notable features of myCred for Event Espresso 4

* Allow users to pay for events using points.
* Badge support.
* Translation ready.

== Installation ==

1. Go to Plugins > Add New.
2. Under Search, type myCred for Event Espresso
3. Find myCred for Event Espresso and click Install Now to install it
4. If successful, click Activate Plugin to activate it and you are ready to go.
